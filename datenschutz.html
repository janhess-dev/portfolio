Hosting

This website is hosted via Vercel. When the site is accessed, technical access data may be processed, especially IP address, date and time of access, requested files, user agent, referrer, and status information. The processing is used to provide, secure, and technically stabilize the website.

Legal basis: Art. 6(1)(f) GDPR. Further information about Vercel's data processing can be found in Vercel's privacy and DPA documents.

Contact form

If you use the contact form, the data you enter there is processed to handle your request. This usually includes your name, email address, and the content of your message.

Legal basis: Art. 6(1)(b) GDPR if your request is related to a potential contract or project request, otherwise Art. 6(1)(f) GDPR based on the legitimate interest in processing incoming inquiries.

Email delivery via Mailgun

The form data is forwarded to the email service Mailgun so that the request can be delivered by email. In particular, the entered name, email address, and message content are processed for this purpose.

It cannot be ruled out that personal data is processed on servers outside the EU/EEA in this context. Mailgun provides its own contractual and data protection information for such processing.

Storage duration

Personal data from contact requests is stored only as long as necessary to process the request and any follow-up communication. Technical log data is generally stored only for as long as required for secure operation and troubleshooting.

Saved language setting

This website stores the language you select locally in your browser so that the chosen language can be retained on future page visits. This information is not used for profiling or marketing.

Fonts and external resources

This website currently does not load external web fonts from third-party servers. The typography is displayed via locally available system fonts.

Analytics and tracking

This website currently does not use analytics, marketing, or tracking tools of its own.

External links

This website contains links to external websites, for example GitHub. If you click such a link, the privacy policies of the respective external providers apply from that point onward.
