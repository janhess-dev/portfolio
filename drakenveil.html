Overview

What this project is about

Drakenveil is a dark fantasy game prototype built in Unity. At its current stage, the project focuses on building a solid structural foundation through menu design, scene transitions, and responsive user feedback. The goal is to create a clear and scalable base before moving into the core gameplay loop.

Implemented

Current features

  • Main menu structure with multiple navigation options
  • Scene management and basic screen flow
  • Audio feedback for button interactions
  • Dark fantasy visual direction and menu presentation

Next Steps

Planned direction

  • Core gameplay loop
  • Team-based heal / combat mechanics
  • System-driven progression and balancing
  • Playable prototype with clearer interaction feedback

Focus

What I wanted to explore

I wanted to explore how a game project can be structured from the beginning in a way that feels clear, expandable, and responsive. Instead of jumping straight into gameplay, the focus here was to understand scene flow, UI behavior, and the overall user experience around interaction.

Learning

What I learned

This project helped me better understand how to organize scenes, connect interface states, and think more carefully about feedback and flow inside a game environment. It also pushed me to work more deliberately with structure instead of only visuals.

Status

Work in progress

Drakenveil is still in development. The current version is more of a structured prototype than a finished playable experience, but it already reflects the direction of the project: system-focused, atmosphere-driven, and designed as a base for future gameplay mechanics.
